CSS

CSS es un lenguaje de estilos utilizado para controlar la presentación de un documento HTML.

Selectores CSS

  • Selector de etiqueta: p { color: blue; }
  • Selector de clase: .mi-clase { background-color: #f2f2f2; }
  • Selector de ID: #mi-id { font-size: 24px; }

JavaScript

JavaScript es un lenguaje de programación utilizado para agregar interactividad a las páginas web.

Variables y datos en JavaScript

  • Variables: let miVariable = 'Hola mundo';
  • Datos: const miArray = [1, 2, 3];

Tabla de atributos CSS

Atributo Descripción Ejemplo
color Establece el color del texto. p { color: blue; }
background-color Establece el color de fondo de un elemento. body { background-color: #f2f2f2; }
font-size Establece el tamaño de la fuente. h1 { font-size: 36px; }
